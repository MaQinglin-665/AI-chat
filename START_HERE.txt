Xinyu Desktop Pet v1.4 Preview - Start Here
===========================================

This is an MVP preview for early users and contributors, not a mature
commercial product.

Recommended download
--------------------

For normal Windows 10/11 users, download these release assets first:

    Xinyu-AI-Desktop-Pet-Setup-v1.4.0-preview.exe
    SHA256SUMS.txt

The installer copies the project to your user folder, creates Start Menu and
desktop shortcuts, then runs the guided first start.

This first installer is unsigned. Before running it, compare:

    Get-FileHash .\Xinyu-AI-Desktop-Pet-Setup-v1.4.0-preview.exe -Algorithm SHA256
    Get-Content .\SHA256SUMS.txt

If SmartScreen warns about an unknown publisher, continue only when the file
came from this repository release and the SHA256 matches.

Source package path
-------------------

Developers and early testers can use:

    Xinyu-AI-Desktop-Pet-v1.4.0-preview-windows-source-test.zip

From an extracted source package, double-click:

    install_and_start.bat

First model setup
-----------------

Xinyu does not include a cloud model, hosted endpoint, or API key. On first
launch, the chat window shows a model setup wizard when LLM configuration is
missing.

Default provider:

    openai-compatible

Default key env:

    TAFFY_LLM_API_KEY

The real API key is stored only in local .env. Non-secret model settings go to
config.local.json. After saving, the app runs a lightweight model probe and
shows a readable success or failure reason.

Safety defaults
---------------

Desktop observation, screenshots, user file reading, tool calling, and shell
execution stay disabled by default.
