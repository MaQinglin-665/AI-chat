❓ 常见问题

不一定哦。你可以接云端模型（通义千问、DeepSeek 等），也可以切到本地 Ollama 模式。本地模式下，核心聊天能力可以离线使用。

建议走“校验 → 写入 → 重载”这条安全流程。先看 JSON 预览确认，再一键应用到运行中的桌宠；如果个别模块没刷新，再点一次重启即可。

先点“浏览器试听”确认本机扬声器正常，再检查后端 TTS 地址和服务状态。也可以换一个音色，并调整语速/音量后再试一遍。

优先用 HTTP 方式访问页面，先确认 CubismCore 脚本可访问；再检查 `model3.json` 与纹理路径是否按 `autoload.js` 的基准目录解析。

当前优先支持 Windows 10/11。首次启动会自动准备运行环境，之后你可以用一键脚本启动，或走 Electron 桌面方式运行。
