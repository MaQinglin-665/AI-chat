## 可附带的脱敏信息

可以分享：

- 失败命令和最后几行错误。
- `scripts\diagnose-llm-link.ps1 -SoftFail` 的脱敏结果。
- `/api/health` 或 `/doctor` 中不含密钥的摘要。
- 经过遮挡的界面截图。
- 一小段不含隐私的示例对话。

不要分享：

- API key、token、Authorization header。
- 完整 `.env`、`config.local.json` 或真实本地配置。
- 原始 prompt、raw history、长期记忆文件。
- 私有文件路径、桌面截图、聊天记录、语音样本。
- 任何能识别个人、公司、项目机密的信息。

## 当前已知边界

- 项目仍是 MVP / 开源孵化阶段，不是成熟商业产品。
- 首跑路径应优先保证能启动、能聊天、能排查；角色体验还会持续调整。
- 桌面观察、截图上下文、工具调用、shell 执行不应默认开启。
- 高风险能力必须保持可配置、可关闭，并在危险操作前要求用户确认。
- 反馈中如果涉及隐私或安全问题，请优先私下联系维护者，不要直接公开贴细节。
